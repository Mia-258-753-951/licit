🏛️ Gestor de Licitaciones

Sistema para el seguimiento, registro y control de procesos de licitación. Permite desde el simple archivo histórico hasta la gestión documental completa si decides participar en el concurso.

🚀 Funcionalidades:
Registro Flexible: Guarda licitaciones de interés aunque no participes.
Control de Fases: Gestión del ciclo de vida completo de la licitación.
Gestión Documental: Organización de archivos necesarios para cada etapa.
Interfaz Dual: Acceso total mediante línea de comandos (CLI) o interfaz programática (API).

🛠️ Stack Tecnológico:
Core: Python 3.12+ (gestionado con uv)
Base de datos: SQLite (Ligera y sin configuración)
API: FastAPI / Uvicorn
CLI: Typer
